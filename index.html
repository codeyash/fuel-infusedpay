fuel-infusedpay

Easy payment processing for the fuelphp framework.

Download .zip Download .tar.gz View on GitHub

Overview

Using similar syntax and methods as the default auth package in fuel, the InfusedPay package allows you to add and create multiple instances of various payment processors, and then use them with all the details hidden away. The primary calls are as easy as:

Processor::charge(transaction);
Processor::refund(transaction);
Processor::void(transaction);
